# Tool Box
Requisitos
- Adicionar página à uma Box
- Listar determinado grupo de páginas (Box)
- Excluir página em uma Box
- Excluir box

## Arquitetura MVC
Estruturas:
- Controller
- Service
- Repository
- Model

## Banco de Dados
## Tabelas:
